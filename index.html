Liberry

  • Liberry is a desktop library management application.

  • It enables librarians from neighborhood libraries to manage tens of thousands of books and borrowers.

  • It is optimized for librarians to serve borrowers quickly through entering commands into the Command Line Interface (CLI) while still having the benefits of a Graphical User Interface (GUI)
